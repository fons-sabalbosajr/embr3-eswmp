# EMBR3 ESWMP — Front-End

Ecological Solid Waste Management Pipeline for Environmental Management Bureau Region III.

## Tech Stack

- React 19 + Vite 7
- Ant Design 6
- Axios, SweetAlert2, Day.js

## Setup

```bash
npm install
npm run dev
```

Runs on `http://localhost:5173` and proxies `/api` to the backend at `http://localhost:5000`.

## Build

```bash
npm run build
```

Output goes to `dist/`.
